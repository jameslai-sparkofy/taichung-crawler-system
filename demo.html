https://your-username.github.io/your-repo-name/data/permits.json

JavaScript 串接範例：

// 獲取最新建照資料
const response = await fetch('https://username.github.io/repo/data/permits.json');
const data = await response.json();

// 處理資料
data.permits.forEach(permit => {
  console.log(`建照: ${permit.permitNumber}`);
  console.log(`起造人: ${permit.applicantName}`);
  console.log(`基地面積: ${permit.siteArea} m²`);
  
  // 同步到你的CRM系統
  syncToCRM(permit);
});
